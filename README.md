# Course-Recommender-ALS
Модель iALS для рекомендации курсов на основе предыдущих взаимодействий пользователей

## 1. Предобработка данных
- отфильтровал пользователей и курсы, количество взаимодействий для которых меньше 4-ех
- закодировал пользователей и курсы для построения разреженной матрицы
- отобрал топ-3 популярных курса

## 2. Baseline
Бейзлайн MostPopular показал значения метрик `HitRate@3=0.1700` и `MAP@3=0.1090`

## 3. iALS
Применил взвешивание BM25 для матрицы взаимодействий. Подобрал гиперпараметры `n_factors` `alpha` `regularization` `iterations` и получил итоговые метрики `HitRate@3=0.255` и `MAP@3=0.191`, тем самым увеличив метрики, полученные на бейзлайне в  **50%** и **90%** соответственно
